# Стратегия "Цифровой кочевник" / "Digital Nomad"

## Описание стратегии

Стратегия основывается на ["Портфеле лежебоки"](https://assetallocation.ru/lezheboka/), опубликованной Сергеем Спириным 
20 сентября 2010 года в журнале D, №17 (104).

#### Цель стратегии 

Обеспечить доход на уровне чуть выше ставки банковского депозита, при минимальном вмешательстве / 
участии в работе стратегии. Предполагается, что стратегию будут использовать люди, которые не собираются постоянно 
следить за фондовым рынком.

Возможность выйти из стратегии почти в любой момент (когда работает биржа).

#### Суть стратегии 

1. Деньги распределяются равными долями в следующие три актива: 
*  фонд денежного рынка (например [LQDT](https://www.wealthim.ru/products/bpif/wimfl/investment_strategy/) / 
[SBMM](https://first-am.ru/individuals/etf/etf-sbmm) / [AKMM](https://www.alfacapital.ru/individual/bpifs/bpif-akmm))
*  фонд на [Индекс МосБиржи IMOEX](https://www.moex.com/ru/index/IMOEX)
*  золото (GLDRUB_TOM)
2. на каждый актив устанавливается стоп-лосс в размере 10%
3. в случае срабатывания стоп-лосс по какому либо из активов на следующий день деньги от продажи перекладываются в фонд 
денежного рынка
4. в случае если по каким либо причинам выбранный фонд денежного рынка перестает работать, деньги перекладываются в 
 другой фонд денежного рынка от другого провайдера
5. ребалансировка - процессе разработки.


#### Пополнение баланса
Все активы пополняются в равных долях.

#### Выход из стратегии
* с полным выходом из стратегии все очевидно, просто распродаются все активы
* частичный выход из стратегии: каждый из трех активов распродается в равных долях
