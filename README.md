# Стратегия "Цифровой кочевник" / "Digital Nomad"

## Описание стратегии

Долгосрочная стратегия, которая должна обеспечивать:
* минимальные потери во времена нестабильности, финансовых кризисов; 
* доход чуть выше ставки депозита в банке;
* возможность выйти из стратегии почти в любой момент (когда работает биржа).

Стратегия основывается на ["Портфеле лежебоки"](https://assetallocation.ru/lezheboka/), опубликованной Сергеем Спириным 
20 сентября 2010 года в журнале D, №17 (104). В основе упомянутой стратегии: фонд на [облигации](https://www.first-am.ru/individuals/fund/opif-obligatsiy-fond-ros-obligatsiy), [акции](https://www.first-am.ru/individuals/fund/opif-aktsiy-fond-ros-aktsiy) и золото. 
Исходные данные доступны в папке [data](https://github.com/digitalnomadinvestor/digitalnomadstrategy/tree/main/data).

#### Суть стратегии 

1. деньги распределяются равными долями в следующие три актива: 
*  фонд денежного рынка (например [LQDT](https://www.wealthim.ru/products/bpif/wimfl/investment_strategy/) / 
[SBMM](https://first-am.ru/individuals/etf/etf-sbmm) / [AKMM](https://www.alfacapital.ru/individual/bpifs/bpif-akmm))
*  фонд на [Индекс МосБиржи IMOEX](https://www.moex.com/ru/index/IMOEX), например [EQMX](https://www.wealthim.ru/products/bpif/wimfimb/investment_strategy/) (ВТБ), [SBMX](https://www.first-am.ru/individuals/etf/etf-moex) (Сбер)
*  золото - GLDRUB_TOM. Либо, если хочется просто потестировать стратегию на минимальных суммах - 
можно купить фонд на золото ([GOLD](https://www.wealthim.ru/products/bpif/wimfg/investment_strategy/) - от ВТБ, [SBGD](https://www.first-am.ru/individuals/etf/etf-sbgd) - от Сбер, и тд)
2. на каждый актив устанавливается стоп-лосс - 10%, который регулярно перемещается вверх;
3. в случае срабатывания стоп-лосс по какому либо из активов, деньги от продажи перекладываются в фонд 
денежного рынка и там лежат до следующей ребалансировки;
4. в случае если по каким либо причинам выбранный фонд денежного рынка перестает работать, деньги перекладываются в 
 другой фонд денежного рынка от другого провайдера;
5. ребалансировка - процессе разработки.

Отличия от "портефеля лежебоки":
* фонд облигаций заменен фондом денежного рынка
* на активы выставляются стоп лосс, ребалансировка портфеля чаще чем раз в год.


#### Пополнение баланса
Все активы пополняются в равных долях. Если суммы пополнения не хватает для равного распределения по активам, 
на всю сумму пополнения покупаются лоты фонда ликвидности.

#### Выход из стратегии
* с полным выходом из стратегии все очевидно: просто распродаются все активы
* частичный выход из стратегии: каждый из трех активов распродается в равных долях
